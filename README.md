# Network Traffic Analysis & Protocol Investigation

## Project Overview
This project was developed as part of the **Computer Networks** course at **HIT (Holon Institute of Technology)**. In collaboration with my project partner, we performed deep packet analysis to understand network communication flow and traffic patterns.

## Key Learning Objectives & Contributions
* **Traffic Analysis**: Captured and analyzed network traffic using **Wireshark** in a **Linux** environment.
* **Protocol Deep-Dive**: Investigated core protocols including **TCP/IP, HTTP, DNS, ICMP, and ARP**.
* **TCP Handshake**: Analyzed the 3-way handshake process and connection lifecycle.
* **Security & Anomalies**: Identified potential network irregularities and practiced filtering techniques to isolate specific communication streams.
* **Collaborative Development**: Managed the project lifecycle using **Git & GitHub**, ensuring proper documentation and version control.

## Technical Skills Applied
* **Tools**: Wireshark, Linux Terminal, Tshark.
* **Concepts**: OSI Model layers, Packet Headers analysis, IP Addressing, and Subnetting.
